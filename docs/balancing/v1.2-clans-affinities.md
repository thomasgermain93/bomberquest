## Clan Skills — Valeurs calibrées

| Clan | Skill | Effet | Valeur |
|------|-------|-------|--------|
| Ember | Feu Perpétuel | +1 range bombes | +1 tuile (significatif) |
| Storm | Tempo Électrique | -0.3s timer bombe | 15% plus rapide |
| Forge | Peau de Fer | -20% dégâts subis | Réduit frustration stamina |
| Shadow | Voile Doré | +30% coins coffres | Fort pour farm coins |
| Arcane | Résonance Arcanique | +20% chain chance | TODO impl. complète |
| Wild | Instinct Sauvage | +20% vitesse dépl. | Augmente efficacité pathfinding |

### Équilibre inter-clans

**Force relative estimée :**
1. **Shadow** (+30% coins) : meilleur pour farm économique
2. **Storm** (-0.3s timer) : meilleur pour clear speed
3. **Ember** (+1 range) : meilleur pour dégâts AoE
4. **Forge** (-20% dmg) : meilleur pour survie long run
5. **Wild** (+20% spd) : meilleur pour exploration
6. **Arcane** (chain 20%) : potentiel élevé, implémentation partielle

**Risque : Shadow trop fort** pour le farm de coins — surveiller le taux de gain journalier.

| Clan héros ↓ / Ennemi → | Slime | Goblin | Skeleton | Orc | Demon |

## UI Combat — Panneau héros

**Fonctionnalités :**
- Nom court + niveau + état (⚔️ actif / 💨 retraite / 😴 repos / 💀 KO)
- Barre de stamina colorée (vert → orange → rouge)
- Visible en Chasse au Trésor ET Story Mode

**Lisibilité :** ok sur 6 héros, risque de surcharge visuelle avec 8+ héros (cas rare).

## Risques identifiés

| Risque | Probabilité | Mitigation |
|--------|------------|------------|
| Shadow clan trop dominant | Moyen | Réduire coin_bonus à 20% si confirmé |
| Ember range +1 trop fort en early | Faible | Le skill nécessite 2+ Ember dans l'équipe |
| Affinité perçue comme faible | Moyen | Communiquer les bonus via tooltip en UI |
| Pool skills = héros trop puissants | Faible | Les nouveaux skills ne cumulent pas les anciens |

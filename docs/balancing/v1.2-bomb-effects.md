## Impact gameplay attendu

**Actuel (v1.2) :** effets visuels uniquement (couleurs d'explosion), gameplay effects partiels.

**Roadmap v1.3+ :**
- burn : DoT sur ennemis dans la zone (1-2 HP/s pendant durée)
- chain_arc : propagation à 1 ennemi adjacent (+20% dégâts réduits)
- void_drain : +3% stamina max du héros par ennemi touché
- vine_snare : ralentissement enemy.speed × 0.7 pendant 0.9s
- shockwave : knockback ennemi de 1 tuile
- mana_pulse : bomb cooldown -15% pendant 2s

## Architecture Clan Bomb System (#169)

Le système est data-driven et extensible :

```typescript
// Ajouter un nouveau clan = ajouter une entrée dans :
// 1. CLAN_BOMB_STYLES (clanBombSystem.ts)
// 2. CLAN_EXPLOSION_EFFECTS (clanBombSystem.ts)
// 3. CLAN_SKILLS (clanSystem.ts)
// 4. CLAN_ENEMY_AFFINITY (clanSystem.ts)
// Aucune modification du moteur de jeu requise.
```

**Compatibilité rétrograde :** bombes sans famille = style/effet par défaut (noir classique, orange).
